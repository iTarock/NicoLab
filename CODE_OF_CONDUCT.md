# Code of Conduct

NicoLab follows a simple rule: be clear, respectful and useful.

## Expected Behavior

- Be kind and professional.
- Assume good intent.
- Keep feedback technical and actionable.
- Respect privacy and operational security.

## Unacceptable Behavior

- Harassment or personal attacks.
- Publishing sensitive information.
- Encouraging unauthorized access.
- Spamming issues or pull requests.

## Enforcement

iTarock may close, edit or remove contributions that violate this code of conduct or create security risk.
